# linktree
C'est une page de lien (linktree) un peu troll et pas complet au niveau des liens.

## Fonctionnalité
- Mode sombre : un bouton pour passer en mode nuit
- Partage : un bouton pour partager la page
- Animation : effet d'apparition fluide au chargement

## Technologies utilisées
- HTML5 : structure sémantique
- CSS3 : utilisation de variables, keyframes et flexbox
- JavaScript : manipulation du dom pour le thème et le partage

Projet réalisé dans le cadre de mon apprentissage personnel.
